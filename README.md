# AI Pattern Maker

English | [简体中文](README.zh-CN.md)

`ai-pattern-maker` is a Codex skill for turning inspiration, keywords, themes, or reference images into original flat graphic pattern directions.

It helps Codex clarify the idea, offer several style directions, then produce a pattern concept, design notes, image-generation prompt, and practical risk guidance.

## Use Cases

- Flat pattern design
- Visual motif exploration
- Surface graphics
- Poster, apparel, packaging, sticker, badge, or social graphic concepts
- Master or movement style reference selection
- Image-generation prompt writing for original graphic patterns
- Short work titles and design concept copy

## 30-Second Start

```bash
npx skills add https://github.com/JefferyHo/ai-pattern-maker --skill ai-pattern-maker
```

You can also send this to an AI Agent with shell access:

```text
Install ai-pattern-maker. Clone https://github.com/JefferyHo/ai-pattern-maker into ~/.codex/skills/ai-pattern-maker. After installation, check that SKILL.md, agents/openai.yaml, and examples/example-request.md exist.
```

If it is already installed, use this to update:

```text
Update ai-pattern-maker. Go to ~/.codex/skills/ai-pattern-maker, run git pull, then tell me the latest commit.
```

After installation, say this to your Agent:

```text
Use ai-pattern-maker to help me turn "urban renewal and future living" into an original flat graphic pattern. First give me several style directions to choose from.
```

## Manual Install

If you prefer to install without `npx`, clone this repository into your skills directory:

```bash
git clone https://github.com/JefferyHo/ai-pattern-maker ~/.codex/skills/ai-pattern-maker
```

The final installed structure should include:

```text
~/.codex/skills/ai-pattern-maker/SKILL.md
```

Start a new Codex session or reload skills if your environment supports it.
