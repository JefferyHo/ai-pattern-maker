# Example Request

```text
Use ai-pattern-maker to help me turn "urban renewal and future living" into an original flat graphic pattern. First give me several style directions to choose from.
```

```text
使用 ai-pattern-maker，帮我根据“城市更新与未来生活”做一个平面图案设计，先给我几个大师或流派风格参考方向。
```

Expected behavior:

1. Offer several distinct style directions.
2. Wait for the user to choose one.
3. Generate title options, design concept, pattern direction, image-generation prompt, and risk notes.
